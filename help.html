What is a Kanban Board?

The Concept of Kanban was originated in 1940s by Toyota. In Japanese "Kanban" means "Visual Signal" or Visual Card. The Main Purpose of Kanban is to help and improve manufacturing efficiency and create a transparent work process for everybody.

6 Core Practices of Kanban

  1. Visualize the Workflow
  2. Limit Work in Progress
  3. Manging Workflows
  4. Implement explicit policies
  5. Allow Room for Feedback
  6. Always search for improvement
